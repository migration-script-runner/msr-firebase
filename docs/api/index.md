# API Reference

Complete API documentation for MSR Firebase classes, interfaces, and types.

## Core Classes

- **FirebaseRunner** - Main migration runner class extending MigrationScriptExecutor
- **FirebaseHandler** - Database handler implementation for Firebase Realtime Database

## Interfaces

- **FirebaseDataService** - Interface for Firebase data operations
- **EntityService** - Interface for entity CRUD operations

## Services

- **FirebaseDataServiceImpl** - Implementation of Firebase data operations
- **EntityServiceImpl** - Entity management service

## Types

TypeScript type definitions and utilities for Firebase migrations.
